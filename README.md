# Brain-tumor-using-CNN

Brain tumors are dangerous medical conditions that require early detection for effective treatment. MRI scans are commonly used to diagnose brain tumors, but manual analysis can be time-consuming. This project uses Convolutional Neural Networks (CNN) to automatically detect brain tumors from MRI images. The system preprocesses MRI images, applies deep learning techniques, and classifies them into tumor and non-tumor categories. The model is implemented using Python with deep learning libraries and evaluated using accuracy and other performance metrics. This approach helps improve the speed and accuracy of brain tumor diagnosis.
